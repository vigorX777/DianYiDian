# v0.4.0 Liquid Glass UI 改造方案

## Summary

v0.4.0 聚焦 UI 质感重构，不新增打卡类型、不新增统计页、不扩展账号同步。目标是让「点一点」从普通菜单栏工具升级为具有 macOS 26 Liquid Glass 记忆点的轻量打卡体验。

本次采用兼容实现：当前工程最低支持 macOS 14，因此不直接依赖 macOS 26 SDK 的 `glassEffect` API，而使用 SwiftUI Material、半透明层、细描边、高光、柔和阴影和 AppKit `NSPopover` 来实现接近 Liquid Glass 的视觉层次。后续迁移到完整 Xcode 26 / macOS 26 SDK 时，再将兼容层替换为系统级 Liquid Glass API。

## Design Direction

设计关键词：轻、透、液面、可读。

- 轻：不做重型 dashboard，不堆叠复杂卡片。
- 透：窗口和浮层使用磨砂材质，但文本和关键控件保持高对比。
- 液面：进度图标、当月日期表和完成态反馈都围绕“水位”视觉统一。
- 可读：透明效果服务于层级，不牺牲设置项识别和重复使用效率。

## Scope

### 1. 设置窗口玻璃化

- 移除默认 `Form` 外观，改为自定义 SwiftUI 布局。
- 窗口采用透明 titlebar、可拖拽背景和磨砂材质底层。
- 左侧为玻璃场景栏，显示场景图标、名称、启用状态和固定状态。
- 右侧为玻璃编辑面板，保留现有所有配置项：
  - 开机自启
  - 打卡成功提示
  - 达到每日目标时提醒
  - 打卡动效
  - 达标小庆祝
  - 菜单栏显示
  - 场景展示
  - 场景名称、每日目标、今日初始次数
  - 场景图标、主题色、启用、固定、提醒
- 底部保存区改为轻量玻璃工具栏，保留新增场景、保存和状态反馈。

### 2. 右键菜单改为玻璃 Popover

- 右键仍作为打开入口，但不再使用原生 `NSMenu` 作为主要 UI。
- 新增 SwiftUI `StatusMenuView`，通过 `NSPopover` 展示。
- Popover 内容包含：
  - 当前场景标题和今日进度
  - 上次打卡时间
  - 当月水位日期表
  - 场景切换列表
  - 打卡、撤销、重置、设置、退出操作
- Popover 使用超薄磨砂背景、玻璃描边和高光。

### 3. 图标和反馈统一玻璃质感

- 菜单栏图标保留“进度环 + 场景符号”结构。
- 增加更明显的玻璃底盘、高光边缘和完成态光晕。
- 打卡短反馈保持 600ms 内完成，避免干扰。
- 达标小庆祝使用玻璃 popover，不做全屏礼花。

### 4. 当月水位日期表视觉升级

- 保留当前月、周一开头、水位状态和未来日期灰显。
- 日期格改为玻璃小胶囊，水位填充带渐变和顶部高光。
- 今天用细描边强调，达标日期用轻完成态边线。

## Non-goals

- 不新增完成型、时长型打卡。
- 不新增独立历史页、统计图、周报、月报。
- 不新增账号、云同步、模板库。
- 不依赖 macOS 26 SDK，避免破坏当前 SwiftPM 构建链。

## Implementation Plan

1. 新增 SwiftUI 玻璃样式组件和通用颜色映射。
2. 重构 `SettingsView`，保留 `SettingsViewModel` 的现有业务绑定。
3. 调整 `SettingsWindowController`，启用透明 titlebar 和更合适窗口尺寸。
4. 新增 `StatusMenuView`，用 `NSPopover` 替代右键原生菜单主路径。
5. 保留必要的 AppKit 操作桥接，确保打卡、撤销、重置、切换场景、设置和退出都可用。
6. 优化 `StatusIconRenderer`、`MonthCalendarMenuView` 和达标 popover 视觉。
7. 运行构建、CoreChecks、打包和本机菜单栏验证。
8. 同步治理文件并提交。

## Test Plan

- `swift build`
- `swift run DianYiDianCoreChecks`
- `sh Scripts/package-app.sh`
- 启动 `点一点.app`，确认菜单栏项存在。
- 左键点击计数增加。
- 右键点击打开玻璃 Popover。
- Popover 中可执行打卡、撤销、重置、设置。
- 设置窗口可打开，新增/保存场景仍可用。
- 设置窗口在浅色背景下可读。
- 菜单栏多图标模式仍可显示多个场景。

## Acceptance Criteria

- 所有 v0.3.0 功能不回退。
- 设置窗口不再呈现默认 `Form` 观感。
- 右键入口呈现玻璃浮层，而不是普通列表菜单。
- 日期水位表和菜单栏图标视觉语言一致。
- 构建、核心校验和本地打包全部通过。

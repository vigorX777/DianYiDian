# 验收清单

本文件用于记录「点一点」每次开发后的固定验收项。后续功能开发完成后，应在 `开发日志.md` 中记录本清单的执行结果。

## 1. 基础构建验收

- [ ] `swift build` 可以通过。
- [ ] `swift run DianYiDianCoreChecks` 可以通过。
- [ ] `sh Scripts/package-app.sh` 可以生成 `点一点.app`。
- [ ] 生成的 `点一点.app` 可以启动。
- [ ] 启动后进程名为 `DianYiDian`。

## 2. 菜单栏显示验收

- [ ] App 启动后只显示在菜单栏，不显示 Dock 图标。
- [ ] 菜单栏项可见。
- [ ] 菜单栏项显示进度图标。
- [ ] 菜单栏项显示今日次数和每日目标，例如 `1/8`。
- [ ] 深色菜单栏背景下可识别。
- [ ] 浅色菜单栏背景下可识别。
- [ ] 菜单栏空间紧张时仍能确认 App 存在。

## 3. 左键打卡验收

- [ ] 左键点击菜单栏项后，今日次数加 1。
- [ ] 左键点击不会弹出菜单或窗口。
- [ ] 今日次数变化后，菜单栏文本同步刷新。
- [ ] 今日次数变化后，进度图标同步刷新。
- [ ] 单次点击打卡音效能播放。
- [ ] 关闭“打卡成功提示”后，单次点击音效不播放。

## 4. 目标完成音效验收

- [ ] 今日次数首次达到每日目标时，播放目标完成音效。
- [ ] 达标后继续点击，不重复播放目标完成音效。
- [ ] 关闭“达到每日目标时提醒”后，目标完成音效不播放。
- [ ] 修改每日目标后，目标完成判断按新目标生效。

## 5. 右键菜单验收

- [ ] 右键点击菜单栏项后，菜单能稳定打开。
- [ ] 菜单显示当前事项名称。
- [ ] 菜单显示今日次数和每日目标。
- [ ] 菜单显示上次打卡时间，例如“刚刚”“X 分钟前”或“今日未打卡”。
- [ ] 菜单中的“打卡”可以让今日次数加 1。
- [ ] 菜单中的“撤销”可以撤销最近一次打卡。
- [ ] 今日次数为 0 或没有可撤销打卡时，“撤销”不可用。
- [ ] 菜单中的“重置”可以将今日次数重置为今日初始次数。
- [ ] 菜单中的“设置”可以打开设置窗口。
- [ ] 菜单中的“退出点一点”可以退出 App。

## 6. 当月历史日期表验收

- [ ] 右键菜单中显示一个小型当月日期表。
- [ ] 日期表展示当前月份所有日期。
- [ ] 日期表能区分今天、过去日期和未来日期。
- [ ] 每个日期点能显示当天打卡状态。
- [ ] 每个日期点使用水位图效果表达完成比例。
- [ ] 未打卡日期显示空水位。
- [ ] 部分完成日期显示对应比例水位。
- [ ] 达标日期显示满水位或完成态。
- [ ] 超过目标日期仍按满水位显示。
- [ ] 当月历史数据来自本地历史记录和今日状态。

## 7. 设置窗口验收

- [ ] 设置窗口可以打开。
- [ ] 设置窗口使用玻璃面板布局，不再呈现默认 `Form` 外观。
- [ ] 设置窗口文本和控件在浅色背景下可读。
- [ ] 设置窗口底部保存栏不遮挡主要配置项。
- [ ] 设置窗口右侧表单标题和控件按统一标签宽度对齐。
- [ ] 设置窗口多列配置项的列间距一致。
- [ ] 开机自启开关可以保存状态。
- [ ] 打卡成功提示开关可以保存状态。
- [ ] 达到每日目标提醒开关可以保存状态。
- [ ] 打卡动效开关可以保存状态。
- [ ] 达标小庆祝开关可以保存状态。
- [ ] 事项名称可以保存并立即同步到菜单。
- [ ] 每日目标次数限制在 1 到 99。
- [ ] 今日初始次数限制在 0 到 99。
- [ ] 图标样式可以切换。
- [ ] 保存后菜单栏图标和文本立即刷新。
- [ ] 关闭再打开设置窗口时，表单显示上次保存的值。

## 8. 轻提醒与打卡反馈验收

- [ ] 默认场景提醒方式为“不提醒”。
- [ ] 选择“间隔提醒”后，间隔分钟可以保存并在重启后保留。
- [ ] 间隔提醒分钟数支持手动输入任意大于 0 的整数。
- [ ] 未达标且未再次打卡时，间隔提醒会按间隔持续触发。
- [ ] 刚触发过间隔提醒后，未到下一轮间隔前不会重复提醒。
- [ ] 选择“固定时间提醒”后，小时和分钟可以保存并在重启后保留。
- [ ] 固定时间提醒支持新增多个时间点。
- [ ] 固定时间提醒支持删除单个时间点，至少保留一个时间点。
- [ ] 固定时间提醒的每个时间点都可以手动输入小时和分钟。
- [ ] 未达到提醒条件时不会发送提醒。
- [ ] 达到每日目标后不再发送该场景提醒。
- [ ] 通知权限未授权时，设置页显示简短提示，不影响其他保存。
- [ ] 到达提醒条件时，菜单栏会显示轻提示。
- [ ] App 前台运行时，本地通知仍允许展示。
- [ ] 本地通知发送失败时，设置页能显示简短错误提示。
- [ ] 左键打卡后菜单栏图标有短暂反馈。
- [ ] 关闭“打卡动效”后，普通打卡不再播放视觉动效。
- [ ] 首次达标时显示小庆祝 popover。
- [ ] 达标后继续点击不重复触发达标小庆祝。
- [ ] 关闭“达标小庆祝”后，达标不再弹出 popover。

## 9. Liquid Glass 视觉验收

- [ ] 右键点击菜单栏项后打开玻璃 Popover。
- [ ] 右键 Popover 中可见当前场景、今日进度和上次打卡时间。
- [ ] 右键 Popover 中可见当月水位日期表。
- [ ] 当月水位日期表的日期数字居中显示。
- [ ] 当月水位日期表在浅色和复杂背景下有足够可读性。
- [ ] 右键 Popover 中可切换场景。
- [ ] 右键 Popover 中切换场景后菜单保持可见或自动回到可操作状态。
- [ ] 右键 Popover 中打卡、撤销、重置、设置和退出入口可用。
- [ ] 右键 Popover 底部操作按钮有足够点击面积。
- [ ] 菜单栏图标有玻璃底盘和高光，但在复杂壁纸背景下仍可识别。
- [ ] 达标小庆祝使用磨砂玻璃样式。

## 10. 数据与跨天验收

- [ ] App 重启后保留事项配置。
- [ ] App 重启后保留今日次数。
- [ ] 跨天后将前一天记录写入历史。
- [ ] 跨天后今日次数重置为今日初始次数。
- [ ] 跨天后重置当天提醒状态。
- [ ] 手动重置只影响今日次数，不删除历史记录。
- [ ] 历史记录文件为空或损坏时，App 不崩溃。

- [ ] 存在完整 Xcode 工程或可稳定生成 Xcode 工程。
- [ ] Bundle ID 为 `com.vigor.dianyidian`。
- [ ] App 显示名为“点一点”。
- [ ] `LSUIElement` 生效，App 不显示 Dock 图标。
- [ ] App 有正式图标。
- [ ] 可以完成 Archive。
- [ ] 可以完成本地签名。
- [ ] 导出的 App 可以在本机安装并启动。
- [ ] 发布前版本号和构建号已更新。

# 分发流程

当前项目先保留 SwiftPM 本地打包路径。正式 Xcode 工程、签名、Archive 和导出流程需要等待完整 Xcode 环境。

## 当前可用路径

### 构建

```bash
swift build
```

### 核心校验

```bash
swift run DianYiDianCoreChecks
```

### 本地打包

```bash
sh Scripts/package-app.sh
```

产物位置：

```text
.build/app/点一点.app
```

## 正式分发前置条件

- 安装完整 Xcode。
- 执行 `xcode-select` 切换到完整 Xcode。
- 确认 `xcodebuild -version` 可用。
- 准备 App 图标。
- 明确签名身份。
- 明确版本号和构建号。

## 后续正式流程

1. 建立或生成标准 Xcode 工程。
2. 确认 Bundle ID 为 `com.vigor.dianyidian`。
3. 确认 App 显示名为“点一点”。
4. 确认 `LSUIElement` 生效。
5. 配置正式 App 图标。
6. 配置签名。
7. 执行 Archive。
8. 导出本地可安装 App。
9. 在本机安装并验证菜单栏显示、左键打卡、右键菜单、设置窗口和退出流程。
